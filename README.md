🌿 Ceylon Traditional Medicine Center

"Reviving the power of Ayurveda & Hela Wedakama through modern technology"

🧾 Overview

Ceylon Traditional Medicine Center is a full-stack web application built to digitalize and promote Sri Lanka’s traditional healing systems like Ayurveda and Hela Wedakama.

This platform connects patients with experienced doctors, wellness programs, and natural treatments while also integrating modern features like AI-based health suggestions.

✨ Key Features
👤 User Side
🔐 Secure Registration & Login
🩺 Browse Doctors & Treatments
📅 Book Appointments Easily
🧾 View Booking History
⭐ Submit Reviews & Ratings
🤖 AI-based Symptom Checker
📚 Health Articles & Tips


🛠️ Admin Panel

📊 Dashboard with insights
👨‍⚕️ Manage Doctors
🎯 Manage Wellness Activities
📦 Manage Packages
🏨 Manage Accommodations
👥 Manage Users & Bookings
